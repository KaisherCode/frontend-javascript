# Aplicación para resgistar Cursos y Usurarios

Realizar una pequeña aplicación que permita registrar Cursos y Usuarios, para dichos cursos. Los usuarios pueden ser de tipo Alumno (Quiénes se inscribirán en uno más Cursos) y los Profesores que dictarán uno o más Cursos.
